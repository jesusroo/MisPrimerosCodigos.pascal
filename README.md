💻 Mis Primeras Prácticas en Pascal
¡Hola! Bienvenido a este repositorio. Este es un espacio dedicado a almacenar mis primeros pasos en el mundo de la programación, utilizando el lenguaje Pascal.

El objetivo de este proyecto es registrar mi aprendizaje en la lógica de programación, el diseño de algoritmos y el desarrollo estructurado de software para mis estudios universitarios y proyectos personales.

🚀 ¿Qué vas a encontrar aquí?
En este repositorio tengo publicados tres programas prácticos desarrollados desde cero:

🏦 Sistema de Cajero Automático (cajero-automatico.pas): Una simulación de un cajero bancario donde el usuario puede consultar su saldo, depositar dinero y retirar efectivo a través de un menú interactivo.

📊 Calculador de Notas: Una herramienta diseñada para ingresar calificaciones escolares o universitarias y calcular el promedio final de forma automática.

🔢 Suma de Números Negativos (SumaDeNegativos.pas): Un script que lee una lista de números ingresados, detecta cuáles son menores a cero (negativos) y los suma entre sí de manera inteligente.
